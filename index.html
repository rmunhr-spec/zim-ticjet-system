⚙️ One-time setup required — connect Firebase to share tickets across your team
  1. Go to console.firebase.google.com → Create a project (free)
  2. Click ⚙ Project Settings → Your apps → Add web app
  3. Copy the firebaseConfig values into the FIREBASE_CONFIG block at the top of this HTML file
  4. In the Firebase console, go to Firestore Database → Create database → Start in test mode
  5. Save the file and re-open it in your browser
